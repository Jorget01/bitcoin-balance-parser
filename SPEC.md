# Спецификация: Bitcoin Balance Parser

## 1. Обзор проекта
- **Название**: Bitcoin Balance Parser
- **Тип**: GUI-приложение (Windows)
- **Назначение**: Генерация и проверка ссылок bitcoinco.org на наличие положительного баланса
- **Целевая ОС**: Windows 10+

## 2. Функциональная спецификация

### 2.1 Основные функции
1. **Генерация ссылок**: Создание ссылок вида `https://bitcoinco.org/ru/#` + идентификатор
2. **Парсинг страниц**: Загрузка и анализ HTML каждой ссылки
3. **Проверка баланса**: Поиск элемента `<span id="btcBalance">` и извлечение значения
4. **Сохранение результатов**: Запись ссылок с балансом > 0 в txt файл
5. **Параллельная проверка**: Многопоточная проверка для ускорения процесса

### 2.2 Пользовательский интерфейс
- Поле для ввода диапазона идентификаторов (старт-конец)
- Поле для ввода количества потоков
- Кнопка "Старт" для запуска проверки
- Кнопка "Стоп" для остановки
- Лог прогресса с прокруткой
- Статистика (проверено/найдено)

### 2.3 Параметры по умолчанию
- Базовый URL: `https://bitcoinco.org/ru/`
- Файл результатов: `results.txt`
- Количество потоков: 10
- Таймаут запроса: 10 секунд

## 3. Техническая спецификация
- **Язык**: Python 3.x
- **GUI**: tkinter (встроенный)
- **Зависимости**: requests
- **Архитектура**: MVC (Model-View-Controller)

- [ ] Приложение запускается без ошибок
- [ ] GUI отображается корректно
- [ ] Генерация ссылок работает в указанном диапазоне
- [ ] Параллельная проверка работает корректно
- [ ] Ссылки с балансом > 0 записываются в файл
- [ ] Прогресс отображается в реальном времени
- [ ] Остановка работает корректно
